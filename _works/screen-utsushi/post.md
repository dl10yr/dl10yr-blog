# ScreenUtsushi

**ScreenUtsushi** is a simple screen capture tool for Mac and Windows. With intuitive operation, you can quickly capture the screens you need, add annotations, and save them.

- **Full screen**, **window**, and **selection area** capture
- Quick access from system tray
- **Real-time annotations**: rectangles, arrows, text
